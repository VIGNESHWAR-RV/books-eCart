## _List of Features_

* Anonymous user account for Add Cart functionality

* Search Books By  
    + Title
    + ISBN
    + Authors
    + Category

* Filter Book Lists By 
    + Price
    + Published Year
    + Language

* Authentication - LOGIN , SIGNUP  to access user accounts

* STRIPE - Payment process

## _Upcoming Features_

* Password-reset flow
* Payment reesponse in Email
* User-Centric UI improvements
